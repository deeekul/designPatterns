Шаблон: Builder (Строитель)
=======================================================================================================

    Паттерн Builder в Java применяется для создания объектов с большим количеством настраиваемых параметров или полей.
Он позволяет создать объект пошагово, инициализируя только необходимые параметры, и в конечном итоге возвращать
полностью сконфигурированный объект.

=======================================================================================================

    Цель: обеспечить удобство и читаемость создания сложных объектов, а также устранить необходимость заполнения
всех параметров через конструкторы.

=======================================================================================================

    Используется:
1. Когда класс имеет множество параметров: Если у класса слишком много параметров, создать читаемый конструктор
становится сложно. Здесь паттерн Builder делает процесс конфигурирования более явным и удобным.

2. Когда нужно обеспечить неизменяемость объекта**: При использовании паттерна Builder
внутренний составляемый объект часто делается неизменяемым. Это позволяет избежать случайного изменения параметров
объекта после его создания.
